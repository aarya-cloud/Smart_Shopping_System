# Smart_Shopping_System

A Python-based desktop application that helps users find the best grocery deals across multiple platforms using AI-powered analysis.

Built with Tkinter GUI and powered by Google Gemini AI, this app intelligently selects the cheapest options and checks whether your shopping stays within budget.

🚀 Features

🔍 Smart AI Search

Understands user queries like “cake ingredients” or “daily essentials”

💰 Price Comparison

Compares product prices across:

BigBasket

Amazon

Blinkit

🧠 AI Optimization

Uses Google Generative AI to:

Identify relevant products

Pick cheapest options

Calculate total cost

📊 Budget Analysis

Shows if you're within budget

Alerts when you're over budget

🎨 Modern Dark UI

Clean and user-friendly interface built using Tkinter


🧰 Tech Stack

Python

Tkinter (GUI)

JSON

Google Generative AI (Gemini 2.5 Flash)
