# Smart_Shopping_System

A Python-based desktop application that helps users find the best grocery deals across multiple platforms using AI-powered analysis.

Built with Tkinter GUI and powered by Google Gemini AI, this app intelligently selects the cheapest options and checks whether your shopping stays within budget.
